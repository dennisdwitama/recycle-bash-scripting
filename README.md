# FDM Skills Lab - UNIX

## What is this repository about?
This repository hold the required code to work on topics under UNIX sprint throughout the D&A, Software Engg and IT Ops streams.

## Instructions
For each exercise, please save your bash history. i.e. run the following command before starting the exercise
```
history -c && history -w && rm ~/.bash_history
```
Then `cp ~/.bash_history` after completing the exercise to the `solutions/` directory. Clear your history again before starting the next exercise. Please name your history as `exercise{module}`.

## What should I do with this branch?
