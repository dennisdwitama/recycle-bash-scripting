Exercise 5 - More Commands
**Prerequisite for this exercise is a file called `accounts`. Please explore for it in the /student_files directory and its subdirectories**
1. Display the fifth character of each line, in the `accounts` file.
2. Display the fifth character of `FDM Academy`.
3. Display the first 4 characters of each line in the `accounts` file.
4. Display the first and sixth characters of each line in the `accounts` file.
5. Display the first and last names for every person in the `accounts` file.
6. Display the account type and balance for every person in the `accounts` file. Replace the colon with a space.
7. Retrieve Ray Manzarek’s balance.
8. Use the `head` command to print the first 5 lines of `/etc/passwd` file.
9. Use `head` and `sort` commands together to display the first 5 lines in alphabetical order
10. Use `–t` option to define the delimiter and `–k` option to define the field. Display the `accounts` file sorted by last name
11. Use the –n option to numerically `sort` by the balance field.
12. The `–r` option can be used to reverse order. Use cut and `sort` command together to print the account types in reverse order for every user in `accounts` file
13. Use output redirection to store every account holder’s first and last names in alphabetical order in the file called `sortedByName.txt`
14. Change all the digits in the `accounts` file to `*`
15. Use `–d` option to delete the capital letters from the `accounts` file
16. Print first and last names in capital letters for every user in `accounts` file
17. Use output redirection to store `accounts` file as a comma-separated file called `commaStorage.txt`
18. Print only the pin numbers for each user in the `accounts` file without the string `PIN`
